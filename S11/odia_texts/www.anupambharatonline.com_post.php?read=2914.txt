ସାରା ଦେଶରେ ୫ ଜି ସେବାକୁ ଲାଗିବ ଅଧିକ ସମୟ ଦେଶ ନୂଆଦିଲ୍ଲୀ ଭାରତରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ୫ଜି ଟେଲିକମ୍ ସେବା ଆରମ୍ଭ ହୋଇଛି ଏହା ପରେ ଉପଭୋକ୍ତା ଏହି ହାଇସ୍ପିଡ୍ ଟେଲିକମ୍ ସେବା ପାଇବାକୁ ଚାହିଁ ବସିଛନ୍ତି କି ୁ କହୁଛନ୍ତି ଯେ ୫ଜି ସେବା ପାଇବାକୁ ଉପଭୋକ୍ତାମାନଙ୍କୁ ଆଉ କିଛି ବର୍ଷ ଲ ଅପେକ୍ଷା କରିବାକୁ ପଡିବ କାରଣ ଏତେ କମ ସମୟ ମଧ୍ୟରେ ଭାରତ ପରି ବିଶାଳ ଦେଶରେ ଏହି ସେବା ଲାଗୁ କରିବା ସହଜ କଥା ନୁହେଁ ଭାରତରେ ଅଗଷ୍ଟ ମାସରେ ସ୍ପେକଫ୍ରମ ନିଲାମ ଶେଷ ହେବା ପରେ ସେହି ମାସ ୧୮ ତାରିଖରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଏହାର ମାତ୍ର ୬ ସପ୍ତାହ ପରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ଅଳ୍ପ କିଛି ସହରରେ ୫ଜି ସେବା ଆରମ୍ଭ ହୋଇଥିଲା ଏତେ କମ ସମୟ ମଧ୍ୟରେ ବିଦେଶରୁ ଟେଲିକମ ମେସିନ ଆମଦାନୀ କରି ତାହାକୁ କାର୍ଯ୍ୟକ୍ଷମ କରିବା ସହଜ ନୁହେଁ ବୋଲି ବିଶେଷଜ୍ଞମାନେ କହୁଛନ୍ତି କାରଣ ନେଟୱାର୍କ ଉପକରଣ ଆମଦାନୀ କରି ତାହାର ଟେଷ୍ଟିଂ କରିବାକୁ ସମୟ ଲାଗିଥାଏ ଚୀନରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଜର୍ମାନିରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୧୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଛ ଭାରତରେ ସେପେ ମ୍ବର ୨୦୧୦ରେ ୪ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଛ କି ୁ ଏୟାରଟେଲ ଏପ୍ରିଲ ୨୦୧୨ରେ ସେବା ଆରମ୍ଭ କଲା ଓ ସବୁ ସ୍ଥାନରେ ସେବା ପହଞ୍ଚିବାକୁ ୨୦୧୬ ହୋଇଗଲା ଏଣୁ ଭାରତରେ ଯଦି ୨୦୨୨ ଅଗଷ୍ଟରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହେଲା ସବୁ ସ୍ଥାନରେ ସେବା ଆରମ୍ଭ ହେବାକୁ ଅତି କମରେ ୩ରୁ ୪ ବର୍ଷ ଲାଗିବା ଅନୁମାନ କରାଯାଉଛି ପୂର୍ବ ଖବର ପରବର୍ତ୍ତୀ ଖବର ବିଶ୍ୱର ସବୁଠାରୁ ଅପରିଷ୍କାର ବ୍ୟକ୍ତିଙ୍କ ପରଲୋକ ପୁଣି ମହଙ୍ଗା ହେଲା ଅଟା

ସାରା ଦେଶରେ ୫ ଜି ସେବାକୁ ଲାଗିବ ଅଧିକ ସମୟ ଦେଶ ନୂଆଦିଲ୍ଲୀ ଭାରତରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ୫ଜି ଟେଲିକମ୍ ସେବା ଆରମ୍ଭ ହୋଇଛି ଏହା ପରେ ଉପଭୋକ୍ତା ଏହି ହାଇସ୍ପିଡ୍ ଟେଲିକମ୍ ସେବା ପାଇବାକୁ ଚାହିଁ ବସିଛନ୍ତି କି ୁ କହୁଛନ୍ତି ଯେ ୫ଜି ସେବା ପାଇବାକୁ ଉପଭୋକ୍ତାମାନଙ୍କୁ ଆଉ କିଛି ବର୍ଷ ଲ ଅପେକ୍ଷା କରିବାକୁ ପଡିବ କାରଣ ଏତେ କମ ସମୟ ମଧ୍ୟରେ ଭାରତ ପରି ବିଶାଳ ଦେଶରେ ଏହି ସେବା ଲାଗୁ କରିବା ସହଜ କଥା ନୁହେଁ ଭାରତରେ ଅଗଷ୍ଟ ମାସରେ ସ୍ପେକଫ୍ରମ ନିଲାମ ଶେଷ ହେବା ପରେ ସେହି ମାସ ୧୮ ତାରିଖରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଏହାର ମାତ୍ର ୬ ସପ୍ତାହ ପରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ଅଳ୍ପ କିଛି ସହରରେ ୫ଜି ସେବା ଆରମ୍ଭ ହୋଇଥିଲା ଏତେ କମ ସମୟ ମଧ୍ୟରେ ବିଦେଶରୁ ଟେଲିକମ ମେସିନ ଆମଦାନୀ କରି ତାହାକୁ କାର୍ଯ୍ୟକ୍ଷମ କରିବା ସହଜ ନୁହେଁ ବୋଲି ବିଶେଷଜ୍ଞମାନେ କହୁଛନ୍ତି କାରଣ ନେଟୱାର୍କ ଉପକରଣ ଆମଦାନୀ କରି ତାହାର ଟେଷ୍ଟିଂ କରିବାକୁ ସମୟ ଲାଗିଥାଏ ଚୀନରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଜର୍ମାନିରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୧୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଛ ଭାରତରେ ସେପେ ମ୍ବର ୨୦୧୦ରେ ୪ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଛ କି ୁ ଏୟାରଟେଲ ଏପ୍ରିଲ ୨୦୧୨ରେ ସେବା ଆରମ୍ଭ କଲା ଓ ସବୁ ସ୍ଥାନରେ ସେବା ପହଞ୍ଚିବାକୁ ୨୦୧୬ ହୋଇଗଲା ଏଣୁ ଭାରତରେ ଯଦି ୨୦୨୨ ଅଗଷ୍ଟରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହେଲା ସବୁ ସ୍ଥାନରେ ସେବା ଆରମ୍ଭ ହେବାକୁ ଅତି କମରେ ୩ରୁ ୪ ବର୍ଷ ଲାଗିବା ଅନୁମାନ କରାଯାଉଛି ପୂର୍ବ ଖବର ପରବର୍ତ୍ତୀ ଖବର ବିଶ୍ୱର ସବୁଠାରୁ ଅପରିଷ୍କାର ବ୍ୟକ୍ତିଙ୍କ ପରଲୋକ ପୁଣି ମହଙ୍ଗା ହେଲା ଅଟା

ନୂଆଦିଲ୍ଲୀ ଭାରତରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ୫ଜି ଟେଲିକମ୍ ସେବା ଆରମ୍ଭ ହୋଇଛି ଏହା ପରେ ଉପଭୋକ୍ତା ଏହି ହାଇସ୍ପିଡ୍ ଟେଲିକମ୍ ସେବା ପାଇବାକୁ ଚାହିଁ ବସିଛନ୍ତି କି ୁ କହୁଛନ୍ତି ଯେ ୫ଜି ସେବା ପାଇବାକୁ ଉପଭୋକ୍ତାମାନଙ୍କୁ ଆଉ କିଛି ବର୍ଷ ଲ ଅପେକ୍ଷା କରିବାକୁ ପଡିବ କାରଣ ଏତେ କମ ସମୟ ମଧ୍ୟରେ ଭାରତ ପରି ବିଶାଳ ଦେଶରେ ଏହି ସେବା ଲାଗୁ କରିବା ସହଜ କଥା ନୁହେଁ ଭାରତରେ ଅଗଷ୍ଟ ମାସରେ ସ୍ପେକଫ୍ରମ ନିଲାମ ଶେଷ ହେବା ପରେ ସେହି ମାସ ୧୮ ତାରିଖରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଏହାର ମାତ୍ର ୬ ସପ୍ତାହ ପରେ ଅକେଫାବର ପ୍ରଥମ ସପ୍ତାହରେ ଅଳ୍ପ କିଛି ସହରରେ ୫ଜି ସେବା ଆରମ୍ଭ ହୋଇଥିଲା ଏତେ କମ ସମୟ ମଧ୍ୟରେ ବିଦେଶରୁ ଟେଲିକମ ମେସିନ ଆମଦାନୀ କରି ତାହାକୁ କାର୍ଯ୍ୟକ୍ଷମ କରିବା ସହଜ ନୁହେଁ ବୋଲି ବିଶେଷଜ୍ଞମାନେ କହୁଛନ୍ତି କାରଣ ନେଟୱାର୍କ ଉପକରଣ ଆମଦାନୀ କରି ତାହାର ଟେଷ୍ଟିଂ କରିବାକୁ ସମୟ ଲାଗିଥାଏ ଚୀନରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଜର୍ମାନିରେ ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନର ୧୬ ମାସ ପରେ ସେବା ଆରମ୍ଭ ହୋଇପାରିଥିଲା ଛ ଭାରତରେ ସେପେ ମ୍ବର ୨୦୧୦ରେ ୪ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହୋଇଥିଲା ଛ କି ୁ ଏୟାରଟେଲ ଏପ୍ରିଲ ୨୦୧୨ରେ ସେବା ଆରମ୍ଭ କଲା ଓ ସବୁ ସ୍ଥାନରେ ସେବା ପହଞ୍ଚିବାକୁ ୨୦୧୬ ହୋଇଗଲା ଏଣୁ ଭାରତରେ ଯଦି ୨୦୨୨ ଅଗଷ୍ଟରେ ୫ଜି ସ୍ପେକଫ୍ରମ ଆବଣ୍ଟନ ହେଲା ସବୁ ସ୍ଥାନରେ ସେବା ଆରମ୍ଭ ହେବାକୁ ଅତି କମରେ ୩ରୁ ୪ ବର୍ଷ ଲାଗିବା ଅନୁମାନ କରାଯାଉଛି
